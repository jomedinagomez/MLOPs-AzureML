Step 1: Add Azure ML Extension
-------------------------------
1. Open Windows PowerShell (already available)
2. Run this command:
   ```powershell
   az extension add --name ml
   ```
   Note: Azure CLI is already installed on DSVM

Step 2: Login to Azure
-----------------------
1. In PowerShell, run:
   ```powershell
   az login
   ```
2. Follow the browser authentication flow
3. Configure defaults:
   ```powershell

   ```
